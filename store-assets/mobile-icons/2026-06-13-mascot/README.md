# LocateFlow Mascot Store Icons

Canonical source: `apps/mobile/assets/icon.png`.

Use these files when refreshing store metadata:

- `app-store-icon-1024.png` for Apple App Store icon artwork.
- `play-store-icon-512.png` for Google Play store listing artwork.

The native iOS and Android launcher icons are configured in `apps/mobile/app.json`
through `expo.icon`, `expo.ios.icon`, `expo.android.icon`, and
`expo.android.adaptiveIcon.foregroundImage`, all pointing at the same mascot
artwork family.

# AI Memory Index

This folder stores verified project memory.

Rules:
- Only store facts verified from code, config, tests, manifests, migrations, or generated audit reports.
- Do not store secrets.
- Do not store assumptions as facts.
- Use `not verified in code` when evidence is missing.
- Prefer file path and line references.

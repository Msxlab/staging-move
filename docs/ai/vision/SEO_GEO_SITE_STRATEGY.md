## Executive summary

SEO/GEO should use useful, source-backed tools and guides that save into an account. It should not become thin doorway pages, fake local pages, or unreviewed legal/rules content.

10. USPS COA anti-scam guide.

## Hypotheses

- Hypothesis: free tools that save results into an account will convert better than static content.
- Hypothesis: source-backed move-rules pages can become GEO assets if they include source URLs, dates, confidence, and review process.
- Hypothesis: flood-zone or internet-at-new-address tools are safer first MVPs than jurisdiction-heavy rule pages.

## Recommended decisions

- Build one flagship free tool before scaling content.
- Use `tools -> save to account -> Move Command Center` as the conversion loop.
- Use citations, last-verified dates, and confidence labels for any rules or public-source claim.
- Avoid thin ZIP/city doorway pages and unverified legal-sensitive claims.

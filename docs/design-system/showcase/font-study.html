My recommendation · Fraunces

Fraunces sits at the intersection of lux and corporate: it has the softness and italic charm of Instrument Serif, but a sturdier weight axis (400–700), optical sizing, and robust numerals that hold up at 14px on dark backgrounds. It's variable, so we get one file for the whole system — headlines, bill amounts, and statistics all use the same family with different weights.

Runner-up: Newsreader — if you want something quieter and more utilitarian. If you want maximum corporate gravitas: Spectral.
